# Interior Page

## 프로젝트 소개
Interior Page는 셀프 인테리어 및 셀프 시공 콘텐츠를 이미지 중심으로 탐색할 수 있도록 제작한 웹 프로젝트입니다.
로그인 기능을 통해 사용자 접근을 제어하고, `더보기` 버튼을 이용해 추가 이미지 콘텐츠를 동적으로 로드 할 수 있도록 구현했습니다.

## 주요 기능

### 로그인 기능
- 사용자 로그인 기능 구현

### 이미지 콘텐츠 목록
- 셀프 시공 / 셀프 인테리어 카테고리로 구성
- 카드형 이미지 레이아웃 사용
- 인테리어 결과물을 직관적으로 확인 가능

### 더보기 버튼 기능
- 초기 화면에는 일부 콘텐츠만 노출
- `더보기` 버튼 클릭 시:
  - 추가 이미지 콘텐츠가 화면 하단에 로드
  - 사용자가 한 번에 많은 콘텐츠를 보지 않아도 되도록 UX 개선
- `접기` 상태로 다시 줄어드는 구조 구현

### 사용자 친화적 UI
- 이미지 중심의 깔끔한 레이아웃
- 불필요한 요소를 줄여 콘텐츠 집중도 향상
